For this you need to additionally use the 'key' parameter inside the 'max' function, with which you can find the maximum element not by the absolute value, but, for example, by the frequency. In this case the list count() method will be useful.

Code that solves this task fits into a one-line function: 

def most_frequent(data: list[str]) -> str:

    return max(data, key=data.count)

This function will do exactly what you need: count how many times each element enters the input list (key=data.count is responsible for this), and then the max function itself will use this parameter (the frequency) to determine the required element.

There are a lot of other solutions you be able to see after your solution passes all tests successfully. For example: 

from statistics import mode as most_frequent

Or the longer clear way by comparing frequencies in a loop: 

def most_frequent(data: list[str]) -> str:

    answer = 0, ''
    for var in set(data):
        freq = data.count(var)
        if  freq > answer[0]:
            answer = freq, var

    return answer[1]
