Let’s assume we have a list of numbers - [2, 7, 6, 14, -1, 10.5, -1, 5]. If we use the max function, we'll get the number 14 - the maximum element of the list. Code will be as follows: 
def most_frequent(data):
    return max(data)
But we need to find the element with the maximum frequency. What should I do in this case?
For this you need to additionally use the 'key' parameter inside the 'max' function, with which you can find the maximum element not by the absolute value, but, for example, by the frequency. In this case the list.count() method will be useful.
Can I see the full version of the code?
Code that solves this task fits into a one-line function: 
def most_frequent(data):
    return max(data, key=lambda a: data.count(a))
Which can be written even simpler: 
def most_frequent(data):
    return max(data, key=data.count)
This function will do exactly what we need: count how many times each element enters the input list (key=data.count is responsible for this), and then the max function itself will use this parameter (the frequency) to determine the required element.
