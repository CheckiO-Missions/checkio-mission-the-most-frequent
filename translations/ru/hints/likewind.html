Для того, чтобы посчитать количество какого-либо элемента в списке, вам необходимо использовать метод array.count(x)
Он работает следующим образом:
data = ['a', 'a', 'c', 'd', 'e', 'e', 'a', 'b']
data.count('a') --> 3
А как сделать второй список со значениями количества для всех элементов из первого списка?
Отличный способ - использовать list comprehension. Этот способ позволяет не использовать многострочный цикл, а в одну строку создать новый список. В нашем случае это будет выглядеть так: 
        def most_frequent(data):
            n = [data.count(i) for i in data]
Например, применяя этот код к списку из прошлого примера (['a', 'a', 'c', 'd', 'e', 'e', 'a', 'b']), мы получим новый список --> [3, 3, 1, 1, 2, 2, 3, 1], так как вышеуказанный код добавляет в новый список (n) количество вхождений в список для каждого элемента из исходного списка (data).
Что дальше?
Теперь необходимо определить наибольшее значение во втором списке (n). Для подобных случаев в Python есть замечательная встроенная функция 'max'. Применительно к нашему списку ([3, 3, 1, 1, 2, 2, 3, 1]) это работает вот таким образом:
max(n) --> 3
В результате мы получим число, которое покажет, сколько раз самый частовстречаемый элемент находится в первом списке.

Теперь пришло время определить, на какой позиции находится этот самый - максимальный - элемент во втором списке. Именно на этой же позиции находится и сам искомый элемент в первом списке. В этом нам поможет еще один стандартный метод - list.index()
Если мы добавим к нашему коду нужную строчку, то он будет выглядеть вот так: 
        def most_frequent(data):
            n = [data.count(i) for i in data]
            element_index = n.index(max(n))
Если максимальных элементов несколько, как в нашем случае ([3, 3, 1, 1, 2, 2, 3, 1]), то функция вернет индекс самого первого максимального значения. В данном случае - 0. После чего останется только вернуть элемент из первого списка (data) с найденным индексом - element_index.

        def most_frequent(data):
