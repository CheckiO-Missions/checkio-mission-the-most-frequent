Вам необходимо найти элемент, у которого один из параметров - максимальный (в данном случае - частота). Для подобных случаев у Python есть отличная встроенная функция max()
Как именно работает эта функция? Можно пример?
Допустим, у нас есть список чисел - [2, 7, 6, 14, -1, 10.5, -1, 5]. Если мы используем функцию max, то в результате получим число 14 - максимальный элемент списка. Код будет выглядеть так: 
def most_frequent(data):
    return max(data)
Но ведь нам нужно найти элемент с максимальной частотой. Что делать в этом случае?
Для этого нужно дополнительно использовать параметр 'key' внутри функции 'max', с помощью которого можно находить максимальный элемент не по абсолютному значению, а, например, по частоте. В этом случае будет полезен метод list.count()
Я могу увидеть полную версию кода?
Код для решения этой задачи вполне укладывается в однострочную функцию: 
def most_frequent(data):
    return max(data, key=lambda a: data.count(a))
Которую можно записать даже проще: 
def most_frequent(data):
    return max(data, key=data.count)
Данная функция будет делать именно то, что нам нужно - считать, сколько раз каждый из элементов входит в исходный список (за это отвечает key=data.count), а затем сама функция max будет по этому параметру (по частоте) определять нужный нам элемент.
