Для того, чтобы посчитать количество какого-либо элемента в списке, вам необходимо использовать метод list.count(x)
Он работает следующим образом: 
        data = ['a', 'a', 'c', 'd', 'e', 'e', 'a', 'b']
        data.count('a')
data.count('a') --> 3
А как узнать у какого элемента это значение максимальное?
В этом вам поможет стандартная функция max()
Без примера работающего кода - не разобраться...
Код для этого вполне укладывается в однострочную функцию: 
        def most_frequent(data):
            return max(data, key=lambda a: data.count(a))
Которую можно записать даже проще: 
        def most_frequent(data):
            return max(data, key=data.count)
