Nie mam pojęcia jak rozwiązać to zadanie.

musi Pan znaleźć element, którego jednym z parametrów jest maximal (w tym przypadku jest to częstotliwość). Dla tego typu przypadków Python ma świetną wbudowaną funkcję max().

Jak dokładnie działa ta funkcja? Czy może Pan podać przykład?

Załóżmy, że ma Pan listę liczb - [2, 7, 6, 14, -1, 10.5, -1, 5]. Jeśli użyjemy funkcji max, otrzymamy liczbę 14 - maksymalny element listy listy. Kod będzie wyglądał następująco: 

data = [2, 7, 6, 14, -1, 10.5, -1, 5]
print(max(data))

Wyjście:

14

Ale muszę znaleźć element z maksymalną częstotliwością. Co powinienem zrobić w tym przypadku?

W tym celu należy dodatkowo użyć parametru "key" wewnątrz funkcji "max", za pomocą którego można znaleźć maksymalny element nie według wartości bezwzględnej, ale na przykład według częstotliwości. W tym przypadku przydatna będzie metoda list.count().

Czy mogę zobaczyć pełną wersję kodu?

Kod rozwiązujący to zadanie mieści się w jednowierszowej funkcji: 

def most_frequent(data: list[str]) -> str:

    return max(data, key=data.count)

Funkcja ta zrobi dokładnie to, czego Pan potrzebuje: policzy ile razy każdy element wchodzi na listę wejściową (odpowiada za to key=data.count), a następnie sama funkcja max wykorzysta ten parametr (częstotliwość) do określenia wymaganego elementu.

Istnieje wiele innych rozwiązań, które można zobaczyć po pomyślnym przejściu wszystkich testów. Na przykład: 

from statistics import mode as most_frequent

Lub dłuższy, przejrzysty sposób polegający na porównywaniu częstotliwości w pętli: 

def most_frequent(data: list[str]) -> str:

    answer = 0, ''
    for var in set(data):
        freq = data.count(var)
        if  freq> answer[0]:
            answer = freq, var

    return answer[1]
